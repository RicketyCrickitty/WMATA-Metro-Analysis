# DC Metro Analysis

## See [here](https://github.com/RicketyCrickitty/WMATA-Metro-Analysis.git) for report detailing results.

## Data Description

**./data/api_data/** -> contains station distance data in CSV sourced using WMATA API

**./data/csv_format/** -> contains raw data converted from excel format to CSV format which is sourced from WMATA

**./data/ridership/** -> contains enriched data converted from raw data in CSV format
